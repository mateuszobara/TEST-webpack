minimalist boilerplate

created by mateusz obara

Installation

To clone this repo, use:git clone https://github.com/mateuszobara/webpack-boilerplate.git

To install dependencies, use:npm install

Getting started

To run development server, use:npm start

To bundle this project, use:npm run build
